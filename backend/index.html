No-Waste Neko API v1.0 - Documentation

process_login

Description
Validates login information and redirects to homepage if successful.

HTTP request
GET ../../backend/api/process_login.php?username=test&password=test

process_signup

Description
Validates signup information and adds user's login information to database. Redirects to login page.

HTTP request
POST ../../backend/api/process_signup.php?username=neko&password=neko&c_password=neko

get_{item}

Description
Gets objects from the database.

  • Return all preferences that belongs to username in a JSON format.
    GET >../../backend/api/get_preference.php?username=test

  • Return all fridge items that belongs to username in a JSON format.
    GET >../../backend/api/get_fridge.php?username=test

  • Return all saved recipes that belongs to username in a JSON format.
    GET >../../backend/api/get_recipes.php?username=test

add_{item}

Description
Adds objects into the database.

  • Inserts preference that belongs to username into preference table.
    GET >../../backend/api/add_preference.php?username=test&diet=vegan

  • Inserts fridge items that belongs to username into fridge table.
    GET >../../backend/api/add_fridge.php?username=test&productName=fish&productCat=meat&expiryDate=22112023

  • Inserts saved recipes that belongs to username into recipes table.
    GET >../../backend/api/add_recipes.php?username=test&recipeName=Pizza&recipeURL=www.pizza.com

del_{item}

Description
Delete objects from the database.

  • Deletes all preference that belongs to username.
    GET >../../backend/api/del_preference.php?username=test

  • Deletes fridge items that belongs to username.
    GET >../../backend/api/del_fridge.php?username=test&productName=fish&productCat=meat&expiryDate=22112023

  • Deletes saved recipes that belongs to username.
    GET >../../backend/api/del_recipes.php?username=test&recipeName=Pizza&recipeURL=www.pizza.com

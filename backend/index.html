No-Waste Neko API v1.0 - Documentation

process_login

Description
Validates login information and redirects to homepage if successful.

HTTP request
GET http://localhost/No-Waste-Neko-main/backend/process_login.php?username=test&password=test

process_signup

Description
Validates signup information and adds user's login information to database. Redirects to login page.

HTTP request
POST http://localhost/No-Waste-Neko-main/backend/process_signup.php?username=neko&password=neko&c_password=neko

get_{item}

Description
Gets objects from the database.

  • Return all preferences that belongs to username in a JSON format.
    GET http://localhost/No-Waste-Neko-main/backend/get_preference.php?username=test

  • Return all fridge items that belongs to username in a JSON format.
    GET http://localhost/No-Waste-Neko-main/backend/get_fridge.php?username=test

  • Return all saved recipes that belongs to username in a JSON format.
    GET http://localhost/No-Waste-Neko-main/backend/get_recipes.php?username=test

add_{item}

Description
Adds objects into the database.

  • Inserts preference that belongs to username into preference table.
    POST http://localhost/No-Waste-Neko-main/backend/add_preference.php
    parameter value
    username test
    diet Vegan
    intolerence Nuts

  • Inserts fridge items that belongs to username into fridge table.
    POST http://localhost/No-Waste-Neko-main/backend/add_fridge.php
    parameter value
    username test
    productName Chicken
    productCat Meat
    expiryDate 2023-10-23

  • Inserts saved recipes that belongs to username into recipes table.
    POST http://localhost/No-Waste-Neko-main/backend/add_recipe.php
    parameter value
    username test
    imgURL
    recipeName Teriyaki Avocado Salmon
    servingSize
    estCookingTime 45
    recipeURL https://www.eatwell101.com/teriyaki-salmon-bowl-recipe

del_{item}

Description
Delete objects from the database.

  • Deletes all preference that belongs to username.
    GET http://localhost/No-Waste-Neko-main/backend/del_preference.php?username=test

  • Deletes fridge items that belongs to username.
    GET http://localhost/No-Waste-Neko-main/backend/del_fridge.php?username=test&productName=Chicken&productCat=Meat&expiryDate=2023-10-23
    parameter value
    username test
    productName Chicken
    productCat Meat
    expiryDate 2023-10-23

  • Deletes saved recipes that belongs to username.
    GET http://localhost/No-Waste-Neko-main/backend/del_recipe.php
    parameter value
    username test
    imgURL
    recipeName Teriyaki Avocado Salmon
    servingSize
    estCookingTime 45
    recipeURL https://www.eatwell101.com/teriyaki-salmon-bowl-recipe
